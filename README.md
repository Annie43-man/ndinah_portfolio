# 🌌 Ndinah Portfolio

A personal portfolio app built with Python and Flet, featuring a galaxy/space design theme.

**Live project:** [github.com/Annie43-man/ndinah_portfolio](https://github.com/Annie43-man/ndinah_portfolio)

---

## 👩🏾‍💻 About

This is the personal portfolio of **Martin Ndinelao**, an Electrical Engineering student at the University of Namibia (UNAM). The app showcases academic achievements, projects, GitHub contributions, and contact information.

---

## ✨ Features

- Multi-page navigation (About, Skills, Portfolio, Timeline, Projects, Blog, Experience, MATLAB Hub, GitHub)
- Galaxy/space dark theme with aurora colour accents
- MATLAB certificate showcase (7 completed courses)
- Fix-Flow project documentation and GitHub evidence
- Contact information cards

---

## 🛠️ Built With

- Python 3
- [Flet](https://flet.dev/) — Flutter-based UI framework for Python
